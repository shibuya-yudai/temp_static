## BEM記法について
BEMとは？
「Block」「Element」「Modifier」の頭文字。BEM（ベム）と呼びます。

- Block / 構成のルートとなる親要素。
- Element / Blockに付随する子要素。
- Modifier / 状態変化を表す要素。
